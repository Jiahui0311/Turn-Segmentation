## Turn-Segmentation
MATLAB app for turn segmentation from trunk IMU data.

## Overview
This repository contains a MATLAB App Designer application developed for segmenting turns from trunk IMU signals. The app supports interactive inspection and correction of segmented turns.

## Author
Jiahui Wang, PhD Candidate, KU Leuven
